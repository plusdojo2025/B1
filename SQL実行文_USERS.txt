テーブルの名前は大文字or小文字で統一なので
テーブル名は【大文字】で統一します。


1.USERSテーブル
①データベースを作成して使います。
 CREATE DATABASE B1;
 USE B1;

②テーブルを作成します。
 CREATE TABLE USERS(
 id INT AUTO_INCREMENT PRIMARY KEY,      /*id(自動採番)*/
 name VARCHAR(50) NOT NULL,              /*名前*/ 
 email VARCHAR(100) NOT NULL,            /*ログインID(メール)*/
 pw VARCHAR(20) NOT NULL,                 /*パスワード*/
 role VARCHAR(10) NOT NULL,              /*役割*/
 created_at TIMESTAMP,                   /*データを追加した日時*/
 updated_at TIMESTAMP);                  /*データを更新した日時*/
 
 
③テーブルに6件のデータを登録します。(デモデータ)
